High Performance

Utilizes multiprocessing for background downloads, ensuring zero lag in command responsiveness.

Smart Streaming

Intelligent fallback logic: tries direct streaming first, with automatic local download if extraction fails.

Quick Installation

1. Clone & Configure

# Clone the repository

git clone https://github.com/jlaiii/DiscordMusicBot.git

# Navigate to directory

cd DiscordMusicBot

2. Provide Token

Create a file named token in the root directory and paste your Discord Bot Token inside.

3. Run the Automator

The start.py script handles venv creation, dependency installation, and system path setup.

python3 start.py
