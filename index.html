Engineered for Reliability

Advanced features that go beyond simple playback.

Background Prefetching

Automatically downloads the next 6 tracks in the queue to local storage while the current track plays, ensuring zero-latency transitions between songs.

Smart Autoplay

Keeps the music going with a recommendation engine that picks similar tracks based on your history, genre preferences, and word-overlap analysis.

Hybrid Streaming

Dynamically switches between direct FFmpeg network streaming and local file playback based on your environment's capabilities (Deno/FFmpeg).

Bot Commands

Prefix: !
!play <query>

Searches YouTube or plays a direct link. Supports livestream detection.

!autoplay [genre]

Toggles smart autoplay. Optionally specify a genre like 'lofi' or 'rock'.

!skip / !next

Skips the current track and triggers the next buffered song.

!queue

Displays the list of upcoming tracks currently in the buffer.

!pause / !resume

Control playback state without clearing the queue.

!stop

Disconnects the bot and clears all queue data and buffers.

Quick Deployment

# 1. Clone the repository

git clone https://github.com/jlaiii/DiscordMusicBot.git

# 2. Add your token to a 'token' file or set env

echo "YOUR_DISCORD_TOKEN" > token

# 3. Run the installer and bot

python start.py

Self-Healing

The start.py script automatically detects your OS, creates a venv, and installs ffmpeg/deno if missing.

Persistent History

History is saved to autoplay_history.json per-guild to ensure you don't hear the same songs every session.
