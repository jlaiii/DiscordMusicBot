Hybrid Playback

Smart fallback logic that switches between direct streaming and local downloads to ensure 100% uptime even with restrictive APIs.

Multi-Process Worker

Utilizes a dedicated background worker process for downloads, keeping the main bot responsive and lag-free.

Auto-Dependency

Integrated bootstrap script that automatically handles venv, ffmpeg, and deno installation.

Command Reference

Prefix: !

!play <query>

Plays audio from a URL or YouTube search.

!skip / !next

Skips the current track in the queue.

!queue

Displays all upcoming tracks.

!pause / !resume

Toggles playback state.

!stop

Clears the queue and disconnects the bot.

Quick Setup

1

Configure Token

Create a file named token in the root directory and paste your Discord Bot Token, or set the DISCORD_TOKEN environment variable.

2

Run the Bootstrap

Execute the start script. It will automatically create a virtual environment and install dependencies like ffmpeg and yt-dlp.

python start.py
